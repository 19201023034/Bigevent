# Content-Optimierung für LED-Wände: So bringen Sie Ihre Inhalte zum Strahlen

Eine P1.9 LED-Wand entfaltet ihre Wirkung nur mit dem richtigen Content. Hier erfahren Sie, wie Sie Ihre Daten optimal vorbereiten.

## 1. Die richtige Auflösung berechnen

Erstellen Sie Content immer in der nativen Auflösung der Wand:
`Auflösung (px) = Breite (mm) / Pixelabstand (mm)`

## 2. Bildwiederholrate & Moire-Effekt

Unsere Displays bieten eine hohe Refresh-Rate (3840 Hz - 7680 Hz). Das verhindert den unschönen Moire-Effekt auf Fotos und Videos, die Ihre Besucher mit dem Smartphone machen.

## 3. Formate

Nutzen Sie bevorzugt MP4 (H.264) oder MOV (Apple ProRes) für ruckelfreie Wiedergabe auf unseren Media-Servern.

| Parameter | Empfehlung für P1.9 |
|-----------|---------------------|
| Format | MP4 (H.264 / H.265) oder MOV |
| Framerate | 50fps oder 60fps |
| Farbraum | Rec.709 oder HDR10+ |
| Audio | Separat oder embedded |
